# discord bot boilerplate

A simple TypeScript boilerplate for a Discord bot.

## Setup

- clone the repo

```
git clone https://github.com/SeqSEE/discord-bot-boilerplate.git
```

- install the dependencies

```
cd discord-bot-boilerplate
```

```
npm install
```

- build the project

```
npm run build
```

- start the bot

```
npm run start
```
